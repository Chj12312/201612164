    Problem 1

    Write a C program following the directions/requirements below.

    • Declare two integer variables, a and b, and initialize them to 5 and 3 each.
    • Write a code that does "a-b" and prints out the result.
    • Write a code that does "a/b" and prints out the result. The result must be a real number.

    (정수형 변수 a와 b를 5와 3으로 초기화한 후, 차(a-b)와 몫(a/b)을 구하는 프로그램을 작성하시오. 단 몫은 실수로 출력한다)

    Problem 2

    Write a C program following the directions/requirements below.

    • It is known that the gravity at the Moon is just 17% of the gravity in the Earth.
    • Declare the relative gravity of the Moon, that is, 17% as a constant. 
      • You can use either the literal constant or the symbolic constant.
      • FYI, you can define a literal constant using the "#define" pre-processor directive.
      • FYI, you can define a symbolic constant using the "const" keyword.
      • In any case, It is highly recommended to use Upper Characters for a constant identifier
      • If a man's weight is 65Kg in the Earth, what it would be in the Moon ? Write a code that computes and prints it

    (지구에서의 몸무게 65kg을 달에서의 몸무게로 변환시키는 프로그램을 작성하시오. 단, 달의 중력은 지구 중력의 약 17%이며, 달의 중력 17%를 상수 선언하여 계산한다. )
